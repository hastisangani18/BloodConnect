# BloodConnect

BloodConnect is a Full Stack Blood Donation Platform that connects blood donors with patients during emergencies.

## Features
- Donor Registration
- Blood Search by Group & City
- Emergency Request Board
- Admin Dashboard
- Secure Admin Login
- MySQL Database Integration
- WhatsApp SOS Support

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL
